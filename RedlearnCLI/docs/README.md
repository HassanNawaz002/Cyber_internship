RedLearnCLI

RedLearnCLI is a modular, educational offensive security payload learning framework.
It helps learners understand how common web attacks work and how defensive mechanisms respond.

Features:
- Modular CLI-based design
- JSON-based payload storage
- Supports XSS, SQLi, and Command Injection
- Educational explanations for each payload

Usage:
python redlearn.py --type xss
python redlearn.py --type sqli
python redlearn.py --type cmdi

Ethical Use:
This tool is strictly for learning and defensive research purposes only.
